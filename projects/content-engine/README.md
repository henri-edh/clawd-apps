## Usage

```bash

node blog.mjs --type comparison --product mevo_plus --competitor trackman

# Generate a how-to guide
node blog.mjs --type howto --topic "home-simulator-budget"

# Generate a buying guide
node blog.mjs --type guide --category "budget-launch-monitor"
```
